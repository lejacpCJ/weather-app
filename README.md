# Weather App (React + Vite)

Simple weather UI built with React + Vite. Fetches current weather from OpenWeatherMap and displays temperature, humidity, wind speed and an icon.

## Features

- Search by city
- Displays temperature (Â°C), humidity, wind speed and weather icon
- Uses OpenWeatherMap API
- Built with Vite for fast dev and build

## Requirements

- Node.js 16+
- npm
- OpenWeatherMap API key

## Setup

1. Install deps

```bash
npm install
```

2. Add API key

- Create a `.env` with:

```bash
VITE_APP_ID=your_openweathermap_api_key
```

3. Run dev server

```bash
npm run dev
```

4. Build for production

```bash
npm run build
```

5. Preview production build locally

```bash
npm run preview
```
